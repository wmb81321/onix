# Running a validator node

Validator nodes secure Tempo by validating blocks and participating in consensus.

:::info
The active validator set is currently permissioned. If you are interested in becoming a validator, please [get in touch](mailto:partners@tempo.xyz) with the Tempo team. See [Initial setup](/guide/node/validator-setup) for technical details.
:::
